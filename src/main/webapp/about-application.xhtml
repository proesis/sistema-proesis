Este aplicativo desenvolvido por Danilo Souza Almeida (professordanilo.com.br), com recursos web 2.0, para prover as contabilidades o uso de todo o poder da computação na nuvem. O Escritório Virtual foi pensado para auxiliar as contabilidades a melhorar e manter exelência no atendimento ao seus clientes. Criado na linguagem de programação Java com framework JSF e Primefaces.
