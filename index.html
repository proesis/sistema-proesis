Proesis

Sistema Proesis foi concebido através das dificuldades que os professores enfrentam na hora de criar e gerir suas avaliações.

Inicialmente o sistema foi pensado para suportar um banco de questões de larga escala, abrangendo os mais diversos assuntos e temas de cada disciplina cadastrada.

O sistema conta com um modelo de categoria infinito, onde pode-se adicionar subcategorias a qualquer categoria e esta também receber novas categorias.

Para o futuro está sendo pesquisado uma maneira fácil de utilizar a câmera de um smartphone ou próprio computador para escanear o gabarito, de questões de multipla-escolha, e identificar as respostas marcadas pelo aluno. O grande desafio para o futuro é permitir que o sistema possa exportar os dados lidos das avaliações de forma que outros sistemas possam importar.
